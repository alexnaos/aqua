# 🐠 AquaControl ESP32

Проект системы управления аквариумом на базе ESP32 с использованием библиотек GyverHub и GyverControl.

## Функционал

- 💡 **Управление освещением** - включение/выключение и регулировка яркости
- ⚙️ **Управление оборудованием** - помпа/фильтр, нагреватель
- 🌡️ **Контроль температуры** - автоматическое поддержание заданной температуры с гистерезисом
- 📱 **Веб-интерфейс** - удобное управление через браузер или мобильное приложение

## Требования

- Плата ESP32 (например, ESP32 DevKit V1)
- PlatformIO IDE (расширение для VS Code или отдельная установка)
- Библиотеки (устанавливаются автоматически):
  - GyverHub
  - GyverControl
  - ArduinoJson
  - PubSubClient

## Установка и настройка

### 1. Клонирование проекта

```bash
git clone https://github.com/alexnaos/aqua.git
cd aqua/aqua_espi
```

### 2. Настройка WiFi

Откройте файл `src/main.cpp` и измените строки:

```cpp
const char* ssid = "YOUR_WIFI_SSID";        // Ваше имя сети WiFi
const char* password = "YOUR_WIFI_PASSWORD"; // Ваш пароль WiFi
```

### 3. Настройка пинов (опционально)

В файле `src/main.cpp` можно изменить назначения пинов:

```cpp
#define LED_PIN 2           // Свет
#define PUMP_PIN 4          // Помпа
#define HEATER_PIN 5        // Нагреватель
#define TEMP_SENSOR_PIN 34  // Датчик температуры (ADC)
```

### 4. Компиляция и загрузка

```bash
# Инициализация PlatformIO (если нужно)
pio init

# Сборка проекта
pio run

# Загрузка в ESP32
pio run --target upload

# Мониторинг последовательного порта
pio device monitor
```

Или используйте интерфейс PlatformIO в VS Code.

## Подключение оборудования

### Схема подключения реле:

```
ESP32          Реле модуль
----           -----------
GPIO 2   --->  IN1 (Свет)
GPIO 4   --->  IN2 (Помпа)
GPIO 5   --->  IN3 (Нагреватель)
3.3V     --->  VCC
GND      --->  GND
```

**Внимание!** Реле модули часто работают с инвертированной логикой (LOW = включено).

### Датчик температуры:

Для подключения аналогового датчика температуры используйте pin GPIO 34 (или любой другой ADC pin).

Для цифровых датчиков (DS18B20, DHT22) потребуется добавить соответствующие библиотеки и изменить функцию `readTemperature()`.

## Использование

1. После загрузки прошивки ESP32 подключится к WiFi
2. Если подключение не удалось, будет создана точка доступа `AquaControl_AP` с паролем `12345678`
3. Откройте браузер и перейдите по IP-адресу устройства (см. Serial монитор)
4. Используйте веб-интерфейс для управления аквариумом

## Интерфейс GyverHub

Веб-интерфейс включает:

- **Освещение** - переключатель и слайдер яркости
- **Оборудование** - управление помпой и нагревателем
- **Авто-режим** - автоматическое поддержание температуры
- **Мониторинг** - отображение текущей температуры в реальном времени

## Настройки

В коде можно изменить параметры:

```cpp
float targetTemp = 25.0;    // Целевая температура (°C)
float hysteresis = 0.5;     // Гистерезис (°C)
int lightBrightness = 100;  // Яркость по умолчанию (%)
```

## Расширение функционала

Для добавления поддержки реальных датчиков температуры:

1. Добавьте библиотеку (например, `OneWire` и `DallasTemperature` для DS18B20)
2. Обновите функцию `readTemperature()`
3. Добавьте новые элементы управления в интерфейс GyverHub
